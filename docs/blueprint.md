# **App Name**: Koogle

## Core Features:

- Web Search: Enables users to search the web by entering keywords or phrases into a search bar.
- Search Suggestions: Suggests related search queries as the user types in the search bar to improve search efficiency and discovery.
- Categorized Search: Allows users to filter search results by category, such as news, design, react, ai, and docs, to narrow down search results.
- AI-Powered Semantic Tool: The Koogle search algorithm analyses the intent and context of your query in order to retrieve a superior answer.

## Style Guidelines:

- Primary color: Coral-orange (#FF7F50) for the main call to action and brand highlights. It matches the aesthetic provided in the example image.
- Background color: Soft, desaturated pink (#F8E8EB) to create a gentle and visually pleasing backdrop.
- Accent color: Light grey (#D3D3D3) for subtle UI elements and category tags to maintain a minimalistic look.
- Font: 'Poppins' (sans-serif) for both headlines and body text to maintain a contemporary and precise feel.
- A clean, centered layout to align with the soft pastel aesthetic and modern tech brand style.
- Simple, geometric icons to complement the modern design; use a rounded square icon with an orange-to-pink gradient for the logo.
- Subtle transition animations for UI elements to enhance user experience without overwhelming the minimalist design.
